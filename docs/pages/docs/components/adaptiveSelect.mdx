# Adaptive Select

## Adaptiveness

- Desktop: doesn't block interaction with other page elements when open
- Android & iOS: uses the native device select

```tsx
import AdaptiveModeDemo from "@/shared/adaptiveModeDemo";
import Stack from "@mui/material/Stack";
import AdaptiveSelect, {
  AdaptiveSelectItem,
} from "adaptive-material-ui/components/select/adaptiveSelect";
import { useState } from "react";

export default function () {
  const [adaptiveMode, setAdaptiveMode] = useState("");

  return (
    <AdaptiveModeDemo adaptiveMode={adaptiveMode} onChange={setAdaptiveMode}>
      <AdaptiveSelect adaptiveMode={adaptiveMode} defaultValue={1}>
        <AdaptiveSelectItem value={1}>One</AdaptiveSelectItem>
        <AdaptiveSelectItem value={2}>Two</AdaptiveSelectItem>
      </AdaptiveSelect>
    </AdaptiveModeDemo>
  );
}
```

## API

[MUI Select API](https://mui.com/material-ui/api/select/) with the following changes:

- `MenuProps.disableScrollLock` defaults to `true` on desktop
