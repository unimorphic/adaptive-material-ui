# Adaptive Select

## Adaptiveness

- Desktop: doesn't block interaction with other page elements when open
- Android & iOS: uses the native device select
- iOS: native styling

<Tabs>
  <Tab label="Adaptive">
    ```tsx
    import AdaptiveSelect, {
      AdaptiveSelectItem,
    } from "adaptive-material-ui/components/select/adaptiveSelect";

    export default function Basic() {
      return (
        <AdaptiveSelect defaultValue={1}>
          <AdaptiveSelectItem value={1}>One</AdaptiveSelectItem>
          <AdaptiveSelectItem value={2}>Two</AdaptiveSelectItem>
        </AdaptiveSelect>
      );
    }

    ```

  </Tab>
  <Tab label="iOS">
    ```tsx
    import AdaptiveSelect, {
      AdaptiveSelectItem,
    } from "adaptive-material-ui/components/select/adaptiveSelect";

    export default function Basic() {
      return (
        <AdaptiveSelect adaptiveMode="ios" defaultValue={1}>
          <AdaptiveSelectItem value={1}>One</AdaptiveSelectItem>
          <AdaptiveSelectItem value={2}>Two</AdaptiveSelectItem>
        </AdaptiveSelect>
      );
    }
    ```

    :::warning
    This select uses the native device select so will behave differently depending on the device
    :::

  </Tab>
