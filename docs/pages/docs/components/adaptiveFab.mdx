# Adaptive Floating Action Button (FAB)

## Adaptiveness

- iOS: native styling

```tsx
import AdaptiveModeDemo from "@/shared/adaptiveModeDemo";
import AddIcon from "@mui/icons-material/Add";
import { AdaptiveFab } from "adaptive-material-ui/components/fab";

export default function () {
  return (
    <AdaptiveModeDemo>
      <AdaptiveFab aria-label="add">
        <AddIcon />
      </AdaptiveFab>
    </AdaptiveModeDemo>
  );
}
```

## Background Limitations

The iOS FAB background uses `backdrop-filter` which has [limitations](https://stackoverflow.com/questions/60997948/backdrop-filter-not-working-for-nested-elements-in-chrome)
based on it's parent elements. One solution may be to wrap it in a [Portal](https://mui.com/material-ui/react-portal/)
if it's used as a fixed page element.

## API

[MUI Fab API](https://mui.com/material-ui/api/fab/) with the following changes:

- iOS: `disableTouchRipple` defaults to `true`
- iOS: `size` defaults to `medium`
