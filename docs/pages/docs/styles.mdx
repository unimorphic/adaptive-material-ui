# Styles

## Component Styles

Components with device specific styles can be customized using the related classes prop or theme style override as shown below.
Other styles can be customized using the standard Material UI methods, for example:
[Themed Components](https://mui.com/material-ui/customization/theme-components/),
[Switch Customization](https://mui.com/material-ui/react-switch/#customization)

### iOS Custom Class

```tsx
import { styled } from "@mui/material/styles";
import { switchClasses } from "@mui/material/Switch";
import { AdaptiveSwitch } from "adaptive-material-ui/components/switch/adaptiveSwitch";

const AdaptiveSwitchStyled = styled(AdaptiveSwitch)({
  "&.ios": {
    [`& .${switchClasses.track}`]: {
      backgroundColor: "rebeccapurple",
    },

    [`& .${switchClasses.switchBase}.${switchClasses.checked}`]: {
      color: "purple",

      [`& + .${switchClasses.track}`]: {
        backgroundColor: "purple",
      },
    },
  },
});

export default function () {
  return (
    <AdaptiveSwitchStyled
      adaptiveMode="ios"
      classes={{ ios: "ios" }}
      defaultChecked
    />
  );
}
```

### iOS Custom Theme

```tsx
import { createTheme, ThemeProvider } from "@mui/material/styles";
import { switchClasses } from "@mui/material/Switch";
import { AdaptiveSwitch } from "adaptive-material-ui/components/switch/adaptiveSwitch";
import type {} from "adaptive-material-ui/themes/themeAugmentation";

const theme = createTheme({
  colorSchemes: { dark: true },
  components: {
    AdaptiveSwitch: {
      styleOverrides: {
        ios: {
          [`& .${switchClasses.track}`]: {
            backgroundColor: "pink",
          },

          [`& .${switchClasses.switchBase}.${switchClasses.checked}`]: {
            color: "red",

            [`& + .${switchClasses.track}`]: {
              backgroundColor: "red",
            },
          },
        },
      },
    },
  },
});

export default function () {
  return (
    <ThemeProvider theme={theme}>
      <AdaptiveSwitch adaptiveMode="ios" defaultChecked />
    </ThemeProvider>
  );
}
```
