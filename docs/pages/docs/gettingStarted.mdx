# Getting Started

## Install

1. Install the package

   <PackageManagerTabs command="install adaptive-material-ui" />

1. If you already have Material UI
   1. Verify the installed packages are compatible with the [peerDependencies](https://github.com/unimorphic/adaptive-material-ui/tree/master/package.json)
1. Else
   1. Follow the [Material UI install steps](https://mui.com/material-ui/getting-started/installation/)
