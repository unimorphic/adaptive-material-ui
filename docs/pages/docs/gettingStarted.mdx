# Getting Started

If you're using [Capacitor](https://capacitorjs.com/) or [Next.js](https://nextjs.org/) take a look at the
[example projects](https://github.com/unimorphic/adaptive-material-ui/tree/master/examples)

## Install

1. Install the package

   <PackageManagerTabs command="install adaptive-material-ui" />

1. If you don't already have Material UI follow the [Material UI install steps](https://mui.com/material-ui/getting-started/installation/)

## Theme Provider

Add the `AdaptiveThemeProvider` component to the root of your app, this applies the iOS font & default colors.
If you already had a `ThemeProvider`, replace it with this component and
update your [theme](./styles.html#theme) to target the Adaptive components.

```tsx pure
import { AdaptiveThemeProvider } from "adaptive-material-ui/theme/adaptiveThemeProvider";

export default function App() {
  return (
    <AdaptiveThemeProvider>
      <ChildComponentsGoHere />
    </AdaptiveThemeProvider>
  );
}
```
