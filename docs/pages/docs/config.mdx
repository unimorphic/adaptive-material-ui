# Config

## Adaptive Mode Context

The `AdaptiveModeContext` can be used if you want to use a different user agent
or completely control the adaptive logic yourself.

### Different User Agent

```tsx pure
import { AdaptiveModeContext } from "adaptive-material-ui/adaptiveMode";

<AdaptiveModeContext.Provider value={{ userAgent: req.headers["user-agent"] }}>
```

### Custom Logic

```tsx pure
import { AdaptiveModeContext } from "adaptive-material-ui/adaptiveMode";

let mode = "android";
if (user.name === "personIdoNotLike") {
  mode = "desktop";
}

<AdaptiveModeContext.Provider value={{ mode: mode }}>
```
