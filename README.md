# Adaptive Material UI

This library is a collection of React components based off the [MUI library](https://mui.com/)
which adapt to the current device. See more details in the [overview](http://localhost:3000/docs/index.html)

## Documentation

Get started in the [documentation](http://localhost:3000/docs/gettingStarted.html)
